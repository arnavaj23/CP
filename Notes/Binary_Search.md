#Lower and Upper bound
--------------------------------------
- `lower_bound(v, x) - v.begin()`  
  Finds the first index `ind` such that `v[ind] >= x`.

- `upper_bound(v, x) - v.begin()`  
  Finds the first index `ind` such that `v[ind] > x`.

- `binary_search(v.begin(),v.end(), x) - v.begin()`  
Finds the index `ind` such that `v[ind] = x`.
